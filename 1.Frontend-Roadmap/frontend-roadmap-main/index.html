Frontend Development

Frontend Development is the development of the graphical user interface of a website,through the use of HTML , CSS and Javascript , so that users can view and interact with that website.

  • HTML:

    The hypertext Markup Language or HTML is the standarard markup language for documents designed to be displayed in a web browser.

  • CSS:

    Cascading Style Sheets (CSS) is a stylesheet language used for describing the presentation of a document written in a markup language such as HTML or XML

  1. Pure CSS
  2. Bootstrap (Framework)
  3. Tailwind(Framework)
  4. Sass(Preprocessor)
  • Javascript: Javascript often abbrivated JS , is a programming language that is one of the core technologies of the World Wide Web , alongside HTML and CSS. It is used to add functionality in the website.
  1. Vanilla JS
  2. jQuery(Library)
  3. React(Framework)
  4. Angular(Framework)
  5. Vue(Framework)
  6. Typescript(Preprocessor)
